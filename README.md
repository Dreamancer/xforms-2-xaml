# XForms2XAML

## Zadanie
Cielom projektu je vytvoriť XSL Stylesheet a jednoduchú aplikaci v jazuku JAVA, ktoré umožnia prevod formulárov popísaných pomocou štandardu XForms na desktopové rozhrania popísané pomocou jazyka XAML.
